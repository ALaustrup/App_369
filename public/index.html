Release: latest

Install steps (Android)

  1. Download the APK from this page.
  2. Open the downloaded file in your browser's downloads.
  3. If prompted, allow installs from this source.
  4. Tap Install and launch ASTRA MATRIX.

Troubleshooting

  • If install is blocked, verify unknown app installation is allowed for your browser.
  • If Play Protect warns, confirm the APK came from your trusted domain.
  • Keep your Android version updated for best compatibility and security.
